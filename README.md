# Requirements

- Node.js 8.11.3  
https://nodejs.org/ja/
- TypeScript 2.9.2  
`$ npm install -g typescript`
- yarn 1.7.0  
`$ npm install -g yarn`

# Usage

Execute the following commands.

    $ git clone  
    $ cd kaonashi-webapp  
    $ yarn install  
    $ yarn run start  

And access "localhost:8080" using any browser.
