# Requirements

- Node.js 8.11.3 or later.  
https://nodejs.org/ja/
- TypeScript 2.9.2 or later.  
`$ npm install -g typescript`
- yarn 1.7.0 or later.  
`$ npm install -g yarn`

# Usage

Execute the following commands.

    $ git clone  
    $ cd kaonashi-webapp  
    $ yarn install  
    $ yarn run start  

And access "localhost:8080" using any browser.

# Unit Test

`$ yarn run test`

The coverage report is output to "coverage/".
